## Get the latest [Hyperledger/Besu](https://github.com/hyperledger/besu) Release Binary

### Dependencies
- xmllint
- xmlstarlet
- (MacOS `brew install xmlstarlet` gives you both)
- (Linux/Ubuntu `sudo apt install xmlstarlet libxml2-utils`)

### Usage
```zsh
chmod +x ./get_besu.sh
```
```zsh
./get_besu.sh
```
